To run the program:
make run

To use this JSketch Program:

Select a tool that you want to draw with, Then select a colour then draw.
An item can be selected by using the select tool(Looks like a cursor).
The thickness of an item can be determined by using one of the three different lines at the bottom of the tool pallette.
The eraser and fill button have not been implemented.

Enhancements
Sound is included when the snake dies, or when the snake eats an apple.
A green texture is added to the background.

All icons were taken from "www.findicons.com"
